*** comandos git

	- `git init`
		transforma a pasta atual em um repositório git
	- `git status`
		mostra o estado do repositório atual
	- `git add <ARQUIVO>`
		adiciona o arquivo *ao index*
	- `git commit`
		adiciona os arquivos /que estão no index/ ao *repositório* (finalmente)
	- `git log`
		mostra o histórico do repositório
	- `git checkout <COMMIT>`
		navega entre os diferentes /commits/
